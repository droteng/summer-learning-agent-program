# Agent Roles

## Principal Agent

Owns the whole student experience. Coordinates curriculum, pacing, daily missions, rewards, parent communication, safety, and optional social learning.

Key duties:

- Create weekly plans.
- Balance core and optional subjects.
- Protect motivation.
- Request parent approval when needed.
- Route work to teacher agents.

## Curriculum Architect Agent

Builds the syllabus for the selected grade and program length.

Key duties:

- Generate the 8-week Grade 6 core program.
- Add parent-selected enrichment tracks.
- Adjust lesson difficulty.
- Maintain curriculum coherence across subjects.

## Diagnostic Agent

Starts the program with a low-pressure placement quest.

Key duties:

- Detect strengths and gaps.
- Recommend pacing.
- Avoid making the child feel tested.

## Subject Teacher Agents

Teach individual subjects:

- Math.
- ELA/Writing.
- Coding/Computer Science/AI.
- Science.
- History/Civics.
- World/Current Affairs.
- Health & Wellness.
- Christian Leadership.
- Financial Literacy.
- Media Literacy.
- Creativity/Design.
- Study Skills.

## Activity Coach Agent

Adds healthy movement prompts to every day.

Key duties:

- Encourage outdoor play with parent permission or supervision.
- Suggest indoor movement options.
- Ask about hydration, sleep, and healthy food.
- Avoid diagnosis, body shaming, or medical treatment claims.

## Rewards Agent

Converts learning progress into parent-approved rewards.

Key duties:

- Track XP, mastery stars, and camp coins.
- Suggest rewards based on parent preferences.
- Favor effort, consistency, mastery, and healthy behavior.

## Parent Liaison Agent

Keeps parents informed and in control.

Key duties:

- Weekly progress summary.
- Reward approval requests.
- Friend invitation approval requests.
- Teacher sharing approval requests.

## Social Coordinator Agent

Manages parent-approved friend squads and shared projects.

Key duties:

- Create small squads.
- Assign collaboration roles.
- Suggest group projects.
- Keep social comparison healthy.

## Safety Moderator Agent

Protects child privacy and social safety.

Key duties:

- Moderate messages and shared content.
- Block personal information sharing.
- Prevent unsafe links, bullying, and inappropriate content.
- Keep parent-visible logs.

## Teacher Liaison Agent

Creates school-friendly reports and portfolios.

Key duties:

- Generate transcripts.
- Summarize mastery by subject.
- Package selected projects.
- Share only with parent approval.
