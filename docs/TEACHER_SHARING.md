# Teacher Sharing

## Principle

Parents control all school-teacher sharing.

The product can generate useful educational records, but it should not send child information to a school or teacher without explicit parent approval.

## Shareable Items

- Weekly progress report.
- Subject participation summary.
- Mastery overview.
- Project samples.
- Writing samples selected by parent.
- Coding project selected by parent.
- Certificates and badges.
- Attendance/completion transcript.

## Hidden By Default

- Private health check answers.
- Faith reflections.
- Friend messages.
- Parent reward settings.
- Exact scores unless parent opts in.

## Achievement Export MVP

The app can now prepare a parent-approved transcript package with:

- Certificate preview.
- Attendance transcript by week.
- Shareable badges.
- Parent-approved portfolio highlights.
- School-share summary text.
- Parent checklist for removing private details before sharing.

## Printable Package MVP

Parents can open `/printable-package` from the dashboard to review a combined package and use the browser print dialog to print or save as PDF. The package combines the weekly report, transcript, certificate preview, badges, portfolio highlights, squad project updates, privacy exclusions, and parent checklist.

## Future Formats
