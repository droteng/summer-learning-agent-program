# Agent Handoff

This file is the quick memory for Codex, OpenClaw, or another agent entering the project.

## Current Mission

Build Dr. Spark Academy: a multi-agent summer learning program for U.S. Grade 6 students, expandable to other grade levels.

## Product Direction

The system is organized like a coordinated school:

- Principal Agent: head coordinator.
- Curriculum Architect Agent: creates syllabus and aligns grade level.
- Diagnostic Agent: assesses student level.
- Subject Teacher Agents: teach core and optional subjects.
- Coach Agent: motivates child daily.
- Assessment Agent: checks understanding.
- Rewards Agent: proposes parent-approved rewards.
- Parent Liaison Agent: communicates with parent.
- Social Coordinator Agent: handles friend squads.
- Moderation/Safety Agent: protects child interactions.
- Teacher Liaison Agent: creates school-shareable reports.

## Important Constraints

- Parent control is central.
- Child data and messaging require safety-first design.
- Avoid unsupervised child-to-child private messaging in MVP.
- Health content must educate and encourage, not diagnose or shame.
- Christian content must be opt-in.
- Current affairs must be age-appropriate and bias-aware.
