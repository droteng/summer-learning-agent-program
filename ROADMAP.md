# Roadmap

## Phase 0: Project Foundation

- Create local Git repo.
- Capture product concept and decisions.
- Define Grade 6 curriculum architecture.
- Define agent roles and coordination workflow.
- Decide initial technical stack.
- Connect to GitHub.

## Phase 1: Planning Prototype

- Build command-line or local app prototype of the Principal Agent workflow.
- Generate a sample Grade 6 8-week plan.
- Generate one week of daily missions.
- Generate a parent-facing weekly report.
- Generate a child-facing dashboard mock.

## Phase 2: Web App MVP

- Parent onboarding.
- Child profile and grade selection.
- Grade 6 diagnostic quest.
- Daily mission dashboard.
- Teacher agent lesson generation.
- XP, mastery stars, and camp coins.
- Parent reward approval.
- Weekly parent report.

## Phase 3: Curriculum Depth

- Full 8-week Grade 6 core curriculum.
- Health & Wellness enrichment.
- Christian Leadership enrichment.
- Financial Literacy enrichment.
- Media Literacy enrichment.
- Creativity / Design enrichment.
- Study Skills enrichment.

## Phase 4: Social Learning

- Parent-approved friend invitations.
- Learning Squads.
- Shared projects.
- Parent-visible moderated comments.
- Group reward suggestions.
- Teacher share portal.

## Phase 5: Distribution

- Website subscription flow.
- ClawHub/OpenClaw packaging.
- Desktop-friendly PWA.
- iPad and Android tablet responsive layouts.
- Later native wrappers if needed for deeper parental control integration.
